# Calibration Data System (Final Year Project)

## Overview
This repository contains the code developed for a final year project at Dublin City University. The project focuses on designing a structured system for extracting, storing, and analysing calibration data from certificate files.

## Components
- Python-based import pipeline for Excel and Word certificates
- PostgreSQL database schema
- Master runner for batch processing
- Data extraction and cleaning utilities

## Technologies Used
- Python (pandas, openpyxl, python-docx)
- PostgreSQL
- SQLAlchemy
- Jupyter Notebooks

## Notes
- Raw calibration certificates are not included due to confidentiality
- The system was developed using real data from the National Metrology Laboratory (NML)
